# User Stories

1. As a fan, I want to read a summary of Steve Coogan’s biography and achievements, so that I can learn about his career.

   a. The home page has the actor’s headshot, a short biography, and a shortlist of his notable works and awards.

2. As a fan, I want to read a detailed biography, so that I can learn more about Steve Coogan’s early life, his career, and aspects of his personal life.

   a. The navbar has an <a> tag link to navigate to a detailed biography page.
   b. The home page short biography section has an <a> tag link to navigate to a detailed biography page.

3. As a fan, I want to read a review of Steve Coogan’s autobiography, so that I can decide whether I want to read further about his family life, how he began his career, and personal journey in his own words.

   a. The home page has a short description of his autobiographical book and an external link to a literary book review.

4. As a fan, I want to see photographs, so that I can discover how Steve Coogan’s appearance has changed over the years.

   a. The navbar has an <a> tag link to navigate to the detailed biography page with a photo carousel.
   b. The home page short biography section has an <a> tag link to navigate to the detailed biography page with a photo carousel.

5. As a fan, I want to read a full list of Steve Coogan’s awards, so that I can learn what he has been recognized for.

   a. The navbar has an <a> tag link to navigate to the Awards page.
   b. The home page awards section has an <a> tag link to navigate to the Awards page.
   c. The Awards page details his awards and honors by year.

6. As a fan, I want to watch, read, or listen to some of Steve Coogan’s recent interviews, so that I can get a sense of his real-life personality.

   a. The navbar has an <a> tag link to navigate to the Interviews page.
   b. The home page awards section has an <a> tag link to navigate to the full list of awards page.
   c. The Interviews page has a list of recent interviews.
   d. On the Interviews page, each interview description has an external link to an article, video, or podcast.

7. As a fan, I want to contact the site administrator so that I can make suggestions about additional content.

   a. The navbar has an <a> tag link to open a Contact Form with Name, Email, and Message fields, and Submit and Close buttons.

8. As a media representative, I want to contact the site administrator so that I can provide feedback on the site’s content.

   a. The navbar has an <a> tag link to open a Contact Form with Name, Email, and Message fields, and Submit and Close buttons.
