# Project overview - a tribute website

A tribute website highlighting English actor Steve Coogan's career and achievements.

# Hosting link

(https://peter-mannion.github.io/tribute-to-steve-coogan/)

# Project description

The tribute website is comprised of four html pages. The user navigates to each page through the navigation links in the navigation bar. The user can also open a contact form from the navigation bar to submit name, email, and message to the site administrator.

- Home
- Bio
- Awards
- Interviews

# How to use the site

Navigation

- All html pages are navigable through links in the navigation bar.
- The "Tribute to Steve Coogan Logo" links to the home page.
- The navigation menu collapses to a hamburger menu at a screen width narrower than the bootstrap medium breakpoint (768 px).
- All external links open the target web page in a separate tab.

Home page

- The home page has the actorâ€™s headshot, a short biography, and a shortlist of his notable works and awards.
- The home page short biography section has an <a> tag link that navigates to the detailed biography page with a photo carousel.
- The home page awards and recognition section an <a> tag link that navigates to the full list of awards page.
- The home page has a short description of his autobiographical book and an external link to a literary book review.

Awards page

- The Awards page details his awards and honors by year.
- The 2019 award has an external link to a YouTube video of his acceptance speech.
- The award trophy icons do not display at a screen width narrower than the bootstrap medium breakpoint (768 px).

Interviews page

- The Interviews page has a list of recent interviews.
- Each interview description has an external link to an article, video, or podcast.

Contact menu item

- The navbar has an <a> tag link to open a Bootstrap modal form with Name, Email, and Message fields, and Submit and Close buttons.
